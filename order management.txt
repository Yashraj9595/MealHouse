Order Management Screen – UI reference

Location: MealHouse/lib/features/mess_owner/order_management/

Header
- Back arrow (when navigator can pop)
- Title: "Order Management"
- Actions: Filter icon, Refresh icon

Search
- Placeholder: "Search by customer name or order ID."
- Magnifying glass prefix icon

Status filter tabs (horizontal)
- All, Pending, Confirmed, Ready, Delivered – each with count
- Selected tab: green background, white text and count
- Unselected: white/surface background, dark text

Order card layout
- Row 1: Order ID (e.g. ORD001) left; below it time ago (e.g. 5m ago). Right: status tag (Pending = red with red dot; Confirmed/Ready/Delivered = respective colors).
- Row 2: Circular customer avatar, customer name, green phone icon (right).
- Section "Items": lines "2x Veg Thali" with "₹240" on same row; repeat per item.
- "Total Amount" label with total in green (e.g. ₹280).
- For Pending: Reject (red outline, white bg) and Accept (solid green) buttons.
- For other statuses: Mark Ready / Mark Delivered as applicable.

Footer bar
- Left: "Pending Orders" with count (bold).
- Right: "Est. Prep Time" with value in orange/red (e.g. 15 min).

Colors
- Green: primary (accept, total, phone, selected tab).
- Red: pending status, reject button.
- Orange: Est. Prep Time in footer.
